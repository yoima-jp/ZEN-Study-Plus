# ZEN Study Plus

ZEN Studyの学習体験を向上させるChrome拡張機能です。学習進捗の可視化、ダークモード、通知機能など、効率的な学習をサポートする多くの機能を提供します。

## 🌟 主な機能

### 📊 学習進捗の見える化
- コースページで残りの動画時間、動画数、テスト数、問題数を表示
- 設定で表示項目を個別にカスタマイズ可能
- リアルタイムで学習状況を把握

### 🌙 ダークモード
- ZEN StudyのページをダークテーマでWEBサイトの背景色やテキスト色を自動変換
- iframe内のコンテンツにも対応
- 目の疲労軽減とナイトモード学習をサポート

### 🎥 動画完了通知
- 動画視聴完了時に音声通知とデスクトップ通知を表示
- Discord Webhookによる外部通知にも対応
- 学習の区切りを明確に認識

### 📚 Nプラス教材の自動フィルタ
- 「必修教材のみ」フィルタを自動適用してNプラス教材を非表示
- 設定で有効/無効を切り替え可能
- 学習の優先順位を明確化

### 📈 学習履歴の記録
- 訪問したコースの残り時間や統計データを自動保存
- 設定画面の履歴タブで過去の学習状況を確認
- 学習の振り返りと分析をサポート

### 📋 学習レポート共有
- 学習進捗をスクリーンショットとして保存・共有する機能
- 月間学習計画ページで共有ボタンが利用可能
- 学習成果の可視化と共有

## 📦 インストール方法

### Chrome Web Storeからインストール（推奨）
1. Chrome Web Storeで「ZEN Study Plus」を検索
2. 「Chromeに追加」をクリック
3. インストール完了後、ウェルカムページが自動で開きます

### 開発者モードでインストール
1. このリポジトリをクローンまたはダウンロード
2. Chromeで `chrome://extensions/` を開く
3. 右上の「デベロッパーモード」を有効にする
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. プロジェクトフォルダを選択

## ⚙️ 設定方法

1. 拡張機能アイコンをクリックしてポップアップを開く
2. 「設定」タブで各機能の有効/無効を切り替え
3. Discord通知を使用する場合は、WebhookURLを設定
4. 変更は自動的に保存されます

## 🔧 技術仕様

- **Manifest Version**: 3
- **対象サイト**: `https://www.nnn.ed.nico/*`
- **権限**: 
  - activeTab
  - storage
  - tabs
  - notifications
  - unlimitedStorage
  - scripting
- **ブラウザ**: Chrome/Chromium系ブラウザ

## 💻 開発

### 主要ファイルの説明

#### `content.js`
- ZEN Studyページでの主要機能を実装
- ダークモードの適用
- 学習進捗の表示
- 動画完了通知

#### `additional.js`
- 月間学習計画ページでの追加機能
- スクリーンショット共有機能
- 学習レポート生成

#### `background.js`
- 拡張機能のバックグラウンド処理
- 初回インストール時のウェルカムページ表示
- html2canvasの動的読み込み

#### `popup.js`
- 設定画面のUI制御
- ユーザー設定の保存/読み込み
- 学習履歴の表示

### 開発環境セットアップ

1. リポジトリをクローン
```bash
git clone <repository-url>
cd "ZSP public"
```

2. Chromeの開発者モードで拡張機能を読み込み
3. コードを編集後、拡張機能を再読み込み

## 🐛 トラブルシューティング

### よくある問題

1. **ダークモードが正しく動作しない**
   - ページを再読み込みしてください
   - 他の拡張機能との競合を確認してください

2. **通知が表示されない**
   - ブラウザの通知設定を確認してください
   - 拡張機能の通知権限を確認してください

3. **学習進捗が表示されない**
   - ZEN Studyのコースページで機能をお試しください
   - 設定で表示項目が有効になっているか確認してください
